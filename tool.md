# 如何更新博客表格

- 在 `src/data/personalBlog.ts` 文件中追加数据，需要包含 `author`(作者) / `link`(`github` 链接) / `blog`(博客信息) / `resources`(相关资源)，暂定四个字段；

- 执行 `npm start` 即可；

- 其中 star 数图片可以根据 [shields](https://shields.io/category/social) 网站生成；
